# YOLOV8 Project: Object Detection and Architecture Analysis

Ce dépôt contient des exemples de code pour l'implémentation de YOLOv8 (détection d'objets) et des scripts d'analyse conceptuelle basés sur les figures clés de son architecture (Backbone, Neck, Head) et de son post-traitement (Non-Maximal Suppression - NMS).

## Structure du Projet
- `yolov8_detection.py`: Script d'exemple pour effectuer une détection d'objets avec le modèle YOLOv8.
- `architecture_analysis.py`: Script simulant l'analyse des figures d'architecture et de NMS.
- `requirements.txt`: Liste des dépendances Python.

## Installation
1. Cloner le dépôt :
   ```bash
   git clone [URL_DU_DEPOT]
   cd YOLOV8
   ```
2. Installer les dépendances :
   ```bash
   pip install -r requirements.txt
   ```

## Utilisation
### Détection d'Objets
```bash
python yolov8_detection.py
```

### Analyse Conceptuelle
```bash
python architecture_analysis.py
```
